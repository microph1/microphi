@microphi/debug

Just another wrapper around console.log

@microphi/debug allows log filtering, has a nice timestamp and colors and an hook to manually handle logs regardless whether they'd be displayed on the console or not.

@microphi/list is a wrapper around a Javascript array and a Map allowing O(1) access to the data stored while conserving ordering and ease of inserting data of an array.
